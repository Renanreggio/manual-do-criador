# Manual do Criador

Este projeto é uma **atividade avaliativa prática de HTML, CSS e JavaScript básico**.

O site foi desenvolvido por **Renan Ferreira de Souza (RA: 2760482522032)** e contém:

- Página inicial (`index.html`)  
- Tutoriais (`tutoriais.html`)  
- Projetos (`projetos.html`)  
- Downloads (`downloads.html`)  
- Contato (`contato.html`)  
- Sobre (`sobre.html`)  

---

## Tecnologias utilizadas
- HTML5  
- CSS3  
- JavaScript básico (para dark mode e modal de login)  
- Responsividade para celular e desktop  
- Menu mobile (hambúrguer)  
- Modo escuro universal (persistente em todas as páginas)  

---

## Funcionalidades principais
- Dark mode que se mantém ativo em todas as páginas  
- Menu adaptável para dispositivos móveis  
- Modal de Login / Cadastro  
- Posts recentes, mini produtos e mini galeria  
- Newsletter com cupom de desconto  
- Área de comentários e FAQ  
- Cupom do dia e comunidade  

---

## Estrutura do site
O projeto simula um portal simples para conteúdo relacionado a **fabricação digital, eletrônica e projetos “faça você mesmo”**.

---

### Observações
- Todas as páginas são estáticas e não possuem backend.  
- Imagens utilizadas estão na pasta `/img` do projeto.
