# Manual do Criador

Este projeto é uma **atividade avaliativa prática de HTML e CSS**.

O site foi desenvolvido por **Renan Ferreira de Souza (RA: 2760482522032)** e contém:

- Página inicial (index)
- Tutoriais
- Projetos
- Downloads
- Contato
- Sobre

### Tecnologias utilizadas
- HTML5
- CSS3
- Responsividade para celular e desktop
- Menu mobile (hambúrguer)
- Modo escuro (dark mode)

### Estrutura do site
O projeto simula um portal simples para conteúdo relacionado a fabricação digital, eletrônica e projetos de “faça você mesmo”.

Não utiliza JavaScript avançado, frameworks ou bibliotecas externas — apenas HTML e CSS, conforme proposto para a atividade.

**Observação:**  
As páginas são estáticas e não possuem backend.  
Todas as imagens usadas estão na pasta /img do projeto.
